+===========================================================+
|   如何編譯:
|   terminal(Linux環境或WSL)中輸入以下指令:
|   1.  進入./src/目錄
|   2.  在./src/目錄之下輸入 make
+-----------------------------------------------------------+
|
|   如何執行:
|   1.  進入./src/目錄
|   2.  在terminal(Linux環境或WSL)中輸入以下其中一個指令:
|       2-1. 在./src/目錄之下輸入 make test1
|       或
|       2-2. 在./src/目錄之下輸入 make test2
|       或
|       2-3. 在./src/目錄之下輸入 make test3
|
|   以上3個指令分別將會先編譯，然後將./test目錄之下的
|   test1.c, test2.c, test3.c 作為input argument執行
|
+------------------------------------------------------+
|
|   專案架構
|   ANTLR存放於 ./lib/antlr-4.9.2-complete.jar
|   ./src/目錄存放所有的source code
|   ./test/目錄存放所有的test case
+======================================================+
